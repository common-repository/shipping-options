Adds shipping options to posts, pages or any other selected post type. Full support for Catalog X.

== Description ==

Add a shipping options to any content type with a shipping options fields. Each shipping option has fully customizable name and price.

This plugin will add shipping box to selected post types (e.g. Posts, Pages).

You will be able to enable automatic display on fron-end or you can use [shipping_options] shortcode to place it on certain place.

= Shipping Settings = 

You can set unlimited amount of shipping options.

The shipping settings let you set which post types will support shipping.

You can also set default shipping values.

= Catalog X integration =

Shipping Options plugin is 100% compatible with Catalog X.

== Installation ==

= Minimum Requirements =

* WordPress 3.5 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t even need to leave your web browser. To do an automatic install of Shipping Options, log in to your WordPress admin panel, navigate to the Plugins menu and click Add New.

In the search field type "Shipping Options" and click Search Plugins. Once you’ve found Shipping Options plugin you can view details about it such as the the point release, rating and description. Most importantly of course, you can install it by simply clicking Install Now. After clicking that link you will be asked if you’re sure you want to install the plugin. Click yes and WordPress will automatically complete the installation.

= Manual installation =

The manual installation method involves downloading Shipping Options plugin and uploading it to your web-server via your favourite FTP application.

1. Download the plugin file to your computer and unzip it
2. Using an FTP program, or your hosting control panel, upload the unzipped plugin folder to your WordPress installation’s wp-content/plugins/ directory.
3. Activate the plugin from the Plugins menu within the WordPress admin.

= Upgrading =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
